# 📢 Interactive Subscription Card (PROA Platform)

> Um componente de call-to-action (CTA) interativo, desenvolvido com HTML5 e CSS3, focado em conversão de usuários e gerenciamento de estado nativo.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. Trata-se de um card de convite para inscrição na plataforma PROA, projetado para ser um elemento de destaque em páginas web. 

O componente foi estruturado para fornecer uma experiência de usuário (UX) fluida, permitindo que o visitante se inscreva diretamente ou feche o aviso de forma intuitiva.

## ✨ Destaques e Aprendizados Técnicos

* **Gerenciamento de Estado via CSS:** Implementação de lógica de fechamento do componente através de um `input type="radio"` e uma `label`, demonstrando domínio em técnicas de controle de interface sem o uso de JavaScript.
* **Hierarquia de Conteúdo:** Organização visual clara utilizando classes específicas para títulos (`.text-1`) e chamadas à ação (`.text-2`), otimizando o fluxo de leitura e conversão.
* **Integração de Branding:** Inclusão de logotipos institucionais e links externos configurados para abrir em novas abas (`target="_blank"`), seguindo as melhores práticas de navegação.
* **Design Modular e Flexível:** Estrutura baseada em containers (`.container`, `.box`, `.right`) que facilita a manutenção e a adaptação do layout para diferentes seções de um site.
* **Acessibilidade e Ícones:** Uso da biblioteca **Font Awesome** para fornecer feedback visual de fechamento (`fas fa-times`), melhorando a usabilidade geral do componente.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Lógica de controle nativa.
* **CSS3:** Estilização, gerenciamento de layout e animações de visibilidade.
* **Font Awesome (v5.15.2):** Iconografia para suporte visual.
